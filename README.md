# 🧠 AI vs Human Text Detection

This project detects whether a given text is written by **AI or a human** using Machine Learning and Deep Learning techniques.

---

## 🚀 Features

* Classifies AI-generated vs Human-written text
* Uses Random Forest (ML) and BiLSTM (DL) models
* Simple and easy-to-run code

---

## 📂 Project Files

* `app.py` – main application
* `train_ml.py` – ML model training
* `train_dl.py` – DL model training
* `requirements.txt` – dependencies

---

## ⚙️ Installation

```bash
git clone https://github.com/suraj075/AI-vs-Human-text-detection.git
cd AI-vs-Human-text-detection
pip install -r requirements.txt
```

---

## ▶️ Run the Project

```bash
python app.py
```

---

## 📊 Models Used

* Random Forest
* BiLSTM

---

## ⚠️ Note

Model files (`.pkl`, `.h5`) are not included due to GitHub size limits.
You can train the model using the provided scripts.
